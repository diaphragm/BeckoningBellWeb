# 狩人呼びの鐘Web The Old Hunters

https://bloodborne.9kv.org/

## これなに

名作PS4ゲームBloodborneの協力プレイを募集するためのサービス。
ついでに簡易チャット付き。

## 使用ライブラリ/環境

- Ruby 2.6
- Rails 6
- AssetSync
- Vue.js
- Element UI
- Nginx
- AWS
  - EC2
  - RDS
  - S3
  - VPC
  - Lambda
