# 狩人呼びの鐘Web The Old Hunters

https://bloodborne.live-on.net

## これなに

名作PS4ゲームBloodborneの協力プレイを募集するためのサービス。
ついでに簡易チャット付き。

## 使用ライブラリ/環境

- Ruby 2.6
- Rails 6
- Vue.js
- Element UI
- Nginx
- AWS
  - EC2
  - RDS
  - VPC
  - Lambda
